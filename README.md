## 制作背景
1.Airbnbで良さげな物件を見つける<br>
2.予約(してしまった)<br>
3.キャンセルして別の物件にしたい。<br>
4.キャンセルポリシーを読むと清掃代のみのキャッシュバックであるという事実が発覚<br>
5.ホストに全額キャッシュバックを交渉するも不可能であった。<br>
6.もっと確認しておけばよかったと後悔した。<br>
