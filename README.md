## 概要
### 顧客のアクションを最適化する
1.レビュー閲覧<br>
2.Todoリスト<br>
3.サービスオーナー側の意見<br>
4.ユーザー視点の口コミ<br>
## 制作背景
順番|失敗談|こうすればよかった案
:----:|----|----
01|Airbnbで良さげな物件を見つける|レビューを見る
02|予約(してしまった)|今後の予定をMECEでリストアップ<br>(サジェスト機能あると嬉しい)
03|キャンセルして別の物件にしたい。|一旦その判断の前に寝る。<br>朝、意思決定する<br>夜はWillPowerがカラカラ
04|キャンセルポリシーを読むと清掃代のみのキャッシュバックであるという事実が発覚|キャンセルポリシーで大損した体験談や口コミをレコメンド
05|ホストに全額キャッシュバックを交渉するも不可能であった。|ユーザーがドタキャンすることはいけないこと<br>サービス提供側の気持ちを知りたい
06|もっと確認しておけばよかったと後悔した。|UGCの予約に関するキュレーションサイトを作ろう
